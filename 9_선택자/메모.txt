선택자
다양한 태그를 선택하는 방법의 종류
사용용도 -> css -> 선택자{속성:속성값;}

 ※부모코드 수직나열, 형제코드 수직나열
   ex. 1번 코드가 수직으로 주르륵 나열되는데 그 위로 2번코드가 주르륵 나열됨


★우선순위 선택자 : 동일 속성을 주었을 때 우선적으로 적용되는 선택자
                   id > class > 태그명
★ 더욱 상세하게 작성하면 우선순위를 무시하고 더 강하다
   부모태그>자식 태그 
   <ul li>태그에 스타일 적용시, <il>태그 클래스 스타일을 무시함
 ex, li{color:red;}
 	 .on{color:green;}
 	 ul li{color:blue;}

	<ul><li class="on"></li></ul>
	이면 ul li 스타일이 적용되는 것!!!!!!!!!!!!!!!!11


1. 태그명 선택자 : 모든 같은 태그를 선택하는 방법 -> 한번에 전부 다 같은 것을 명령하는 것
				  적용방법 : <태그명></태그명>     사용방법 : 태그명{속성:속성값;}

2. 클래스 선택자 : 태그의 분류를 지어서 선택하는 방법 -> 분류를 명령하는 것! 일부만 같은 것
				  적용방법 : <태그명 class="클래스명"></태그명>   사용방법 : .클래스명{속성:속성값;}
                ★남과 공유도 가능하고, 여러개 사용가능함 (복수)
				★여러개의 클래스를 하나의 태그에 부여하는 방법 : <태그명 class="클래스명 클래스명"></태그명>

3. 아이디 선택자 : 특정 하나만 골라서 선택하는 방법 -> 하나만 명령하는 것!
                  적용방법 : <태그명 id = "아이디명"></태그명>   사용방법: #아이디명{속성:속성값;}
				★나만 가지고 하나만 사용가능하다 (단수)
