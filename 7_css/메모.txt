캐스케이딩스타일시트 -> CSS
외부스타일시트
html파일 외부에서 스타일시트를 적용하는 방법
사용방법 : <link href="경로" rel="stylesheet" type="text/css">
적용방법 : 선택자{속성:속성값;
		속성:속성값;}

내부스타일시트
html파일 내부에서 스타일시트를 적용하는 방법
사용방법 : <style></style>
적용방법 : 선택자{속성:속성값;
		속성:속성값;}

내부직접스타일시트
html파일 내부에 있는 태그에게 직접적으로 스타일을 적용하는 방법
사용방법 및 적용방법 : <태그명 style="속성:속성값;속성:속성값;"></태그명>
===================================================
CSS 사용방법의 사용율
외부 > 내부 > 내부직접

CSS의 강하게 적용되는 순서
내부직접 > 내부 > 외부
